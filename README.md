This project demostrates how to measure distance using an ultrasonic sensor using Arduino. The sensor calculates the distance of an object by sending ultrasonic waves and receiving the reflected signal.

Components Used:

-Arduino UNO
-Ultrasonic Sensor
-Jumper Wires

Formula Used:

Distance=(Time x Speed of Sound)/2
Distance=(Time x 0.034)/2


Output:

The distance will be displayed in the serial monitor.
