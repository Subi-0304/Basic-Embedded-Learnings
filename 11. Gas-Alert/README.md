This project demonstrates an LPG gas detection system using Arduino. When gas level increases beyond a threshold, LED and buzzer will alert.


Components Used:

-Arduino UNO
-Gas Sensor
-Buzzer
-LED
-Resistors(330 ohm and 4.7kohm)


Output: The LED turns ON when gas is detected as well as the buzzer will produces sound alert, when the gas value exceeds the given limits.
